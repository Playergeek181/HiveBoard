# 3D Printing Dataset

## Overview

This repository contains STL files for multiple mechanical and functional 3D printable components.  
All parts were designed to be printed using **PLA filament**.

The dataset includes:

- Drawers
- Threaded assemblies
- Valves
- Circuit breaker components
- Shock absorber assemblies
- Lamps
- Buttons
- Honeycomb panels
- Keys

---

# Recommended 3D Printing Settings

| Setting | Recommended Value |
|---|---|
| Material | PLA |
| Nozzle Diameter | 0.4 mm |
| Layer Height | 0.20 mm |
| Wall Count | 3 |
| Top/Bottom Layers | 5 |
| Infill | 20% – 40% |
| Print Speed | 50 mm/s |
| Nozzle Temperature | 200–210 °C |
| Bed Temperature | 50–60 °C |
| Cooling Fan | 100% |
| Supports | Only where required |
| Build Plate Adhesion | Skirt or Brim |

# Printing Notes

- PLA was selected for easy printing and good dimensional accuracy.
- Threaded parts should be printed slowly for improved fit and surface quality.
- Functional assemblies may require light sanding or tolerance adjustment depending on printer calibration.
- Supports may be required for overhangs and internal cavities.
- Use 100% cooling for best surface finish with PLA.

| Springs | Vertical |
| Panels | Flat on bed |
| Mechanical Bases | Largest flat face down |
